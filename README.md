# Task 1 - Exploratory Data Analysis

## Objective

Analyze customer churn behavior using IBM Telco Customer Churn Dataset.

## Dataset

IBM Telco Customer Churn Dataset
7043 customer records

## Cleaning Performed

- Fixed TotalCharges datatype
- Handled missing values
- Removed duplicates
- Standardized column names

## Key Insights

1. Month-to-month contracts have highest churn.
2. Higher monthly charges increase churn.
3. Long-tenure customers are more loyal.
4. Fiber internet customers churn more.
5. Senior citizens show slightly higher churn.

- Python
- Pandas
- NumPy
